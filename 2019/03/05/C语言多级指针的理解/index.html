C语言多级指针的理解

Posted on 2019-03-05

对多级指针一些自己的理解!

直接上代码

第yi行

int 型的变量a 为一个普通变量 有自己的地址值&a
对他取地址就相当于降了一个级别 所以需要一个一级指针接收他的地址 即:int *

第er行

int* 型的变量p为一个一级指针变量 有自己的地址值&p
因为p为int *类型 对他取地址就相当于降了两个级别 所以&p对应的的接收类型为 int**

第san行

此时p2为int **类型 p2==&p *p2==p **p2==*p
